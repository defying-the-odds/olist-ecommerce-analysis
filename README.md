# Olist E-Commerce: Sales, Profitability & Customer Intelligence Analysis

## Overview
End-to-end analysis of 100k+ Brazilian e-commerce orders using SQL, Tableau, and business frameworks (RFM, cohort analysis) to identify growth opportunities and profitability drivers.

- Tableau Public
- GitHub
